# Control_Structure_in_Python

# Control Structures in Python

This project contains simple Python programs to demonstrate control structures like *if-else* and *loops*.

## Programs Included

### 1. Even or Odd Check

* Takes a number as input
* Uses if-else to check if the number is even or odd
* Prints the result

### 2. Sum of Numbers from 1 to 50

* Uses a for loop
* Calculates the sum of numbers from 1 to 50
* Displays the final sum

## How to Run

bash
python filename.py


## Purpose

This project is for learning basic control structures in Python.
